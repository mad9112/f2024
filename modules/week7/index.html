# 1. The anatomy of the project

  • Manifest Set details so the app can run
  • Kotlin & Java Files for the core logic of the app
  • GeneratedJava The compiled files (You should not edit these)
  • Res Resources folder for static content like images and strings
  • Gradle scripts to bundle, compile and build the app.

# 3. Cards

Reverse domain name notation Reverse domain name notation is a

convention to avoid name collision

Some examples:

  • java.io.File, a Java class name
  • com.adobe.postscript-font, UTI string for Adobe Systems's PostScript fonts
  • com.apple.ostype, UTI string for Apple's OSType
  • org.omg.CORBA, Java library for CORBA
  • org.w3c.dom, Java library for W3C's DOM
  • org.kde.dolphin.desktop, a desktop file name
