# Week 7 - Android apps

Date

17 Oct 2024

Agenda

  • Creating an Android App

# 1. The anatomy of the project

  • Manifest Set details so the app can run
  • Kotlin & Java Files for the core logic of the app
  • GeneratedJava The compiled files (You should not edit these)
  • Res Resources folder for static content like images and strings
  • Gradle scripts to bundle, compile and build the app.

# 2. Creating an Android Application

We will build a basic counter app that will keep track of how many people are in a room at once by adding 1 as a person enters, and subtract 1 as a person leaves.

# A. New project

On Android Studio, create a new project

Choose an Empty Activity

# B. Covid Quo Vadis

Configure your new project like this

  1. The name of the project. We will build a Covid App.
  2. The package name of the app. You should use your name.
  3. The location for your project files.
  4. The minimum Android version your app needs to run.
  5. When you are done click finish.

# D. Set up the UI

# Using Jetpack Compose

Jetpack Compose tutorial Jetpack Compose is a modern toolkit for building native Android UI. Compose simplifies and accelerates UI development on Android with less code, powerful tools, and intuitive Kotlin APIs.

We write functions to describe what each element in our UI looks like. These functions must be preceded with @Composable.

To build our app we will use the Text (opens new window) Composable and the Button (opens new window) Composable. Also, see the Image (opens new window) Composable for your own challenge.
